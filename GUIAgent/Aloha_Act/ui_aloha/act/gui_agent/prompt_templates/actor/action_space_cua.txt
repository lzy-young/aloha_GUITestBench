1. CLICK: Left click on an element. Required: "position" [x, y]. "value" is not applicable.
2. RIGHT_CLICK: Right click on an element. Required: "position" [x, y]. "value" is not applicable.
3. DOUBLE_CLICK: Double click on an element. Required: "position" [x, y]. "value" is not applicable.
4. TRIPLE_CLICK: Triple click on an element. Required: "position" [x, y]. "value" is not applicable.
5. MOVE: Move mouse to an element without clicking. Required: "position" [x, y]. "value" is not applicable.
6. SCROLL: Scroll at a target area. Required: "position" [x, y]. Set "value" as [dx, dy] where dy < 0 means scroll up and dy > 0 means scroll down.
7. WAIT: Wait for UI to update. "position" is not applicable. Set "value" as wait duration in seconds.
8. INPUT: Type text into the focused field. "position" is not applicable. Set "value" as the text string.
9. KEY: Press a key or key combo. "position" is not applicable. Set "value" as key text, for example "Enter" or "Ctrl+L".
10. DRAG: Drag from one point to another. Use either:
	- "start": [x1, y1] and "end": [x2, y2], or
	- "value": [x1, y1] and "position": [x2, y2].
11. Only return one action each turn, and the action must be from the list above.
